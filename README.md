# Group5_SoftwareQualityVerification
# Software Testing Project: Dev.to Platform

## 1. Project Overview (Tổng quan dự án)
This project focuses on performing comprehensive software quality assurance for three core modules of the Dev.to community platform. The goal is to ensure system reliability, data integrity, and a seamless user experience through functional and API testing.

**Dự án:** Kiểm thử chất lượng phần mềm hệ thống cộng đồng Dev.to, tập trung vào 3 phân hệ cốt lõi nhằm đảm bảo tính ổn định và trải nghiệm người dùng.

## 4. Testing Scope (Phạm vi kiểm thử)
- **Manual Testing:** UI/UX, functional flows, and Markdown editor validation.
- **API Testing:** Authentication, article posting, and real-time commenting via Postman.
- **Tools used:** Excel, Postman, GitHub.
